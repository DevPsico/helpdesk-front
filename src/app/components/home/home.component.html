Help Desk

Olá ! Seja bem vindo ao sistema Help Desk desenvolvido por mim, para treinamento Full Stack.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angulae 12 Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Vercel e o back na Heroku.
O projeto tem como objetido principal abordaqr os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no meu perfil.

Fique a vontade para testar o sistema e deixar seu sugestão.
